# Limitless particles 

##

## This program will utilize mouse input from the user to create unique effects. Inspired by popular character Saturo Gojo

Features

  -  Feature 1

    -    When the user taps either of the mouse keys a particle circle will ripple from the mouse location. Left Mouse Button will create red, and Right Mouse Button will create Blue. Holding either key will cause a circle of rotating particles to form

  -  Feature 2

    -    Dragging the mouse while holding either mouse button will create a particle trail that follows the mouse with the corresponding color.  

  -  Feature 3

    -    Holding both mouse keys will create a purple particle trail. tapping both will create a purple ripple 


## challenges

  - Learning particles in pygame and how to manipulate their physics

  - proper event detection for mouse clicks, holds, and draging.

## Ideal Outcome 

  - implement all three functions press, Hold , and Drag for both mouse keys.

## Minimum Outcome

  - At Least Three functions Press, Hold, and Drag for one key
